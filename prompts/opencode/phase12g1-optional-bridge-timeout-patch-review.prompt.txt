Prompt: Phase 12G1 — Optional Bridge Timeout Patch Review

Purpose:
Future explicit human-approved review of whether to patch the OpenCode bridge timeout. This prompt is for planning only. Do not patch anything without explicit human approval.

Prerequisites (ALL must be true):
- [ ] Human has read docs/phase12/PHASE12G_AGENT_ZERO_TIMEOUT_REVIEW_DIRECT_BRIDGE_FALLBACK.md
- [ ] Human understands the timeout cause: bridge.py uses urllib.request.urlopen(req, timeout=60)
- [ ] Human understands the workaround: use direct bridge or simpler prompts
- [ ] Human explicitly approves reviewing a timeout patch in this prompt
- [ ] Bridge is currently operational
- [ ] Agent Zero is stopped

Hard Boundaries:
- Do not patch bridge.py without explicit human approval.
- Do not change the bind address (must remain 127.0.0.1).
- Do not expose the bridge on LAN.
- Do not modify Agent Zero config.
- Do not change bridge functionality beyond timeout.
- Do not run OpenCode implementation prompts.
- Do not use sudo.
- Do not install packages.

Review Scope:
1. Inspect bridge.py timeout line:
   ~/.config/opencode-bridge/bridge.py line 50: urllib.request.urlopen(req, timeout=60)

2. Evaluate if increasing timeout is safe:
   - Current: 60 seconds
   - Proposed: 120 or 180 seconds
   - Risk: Longer hanging requests if backend is down
   - Benefit: Complex prompts may succeed

3. If human approves patch:
   - Backup bridge.py: cp bridge.py bridge.py.backup-YYYYMMDD-HHMMSS
   - Edit only the timeout value
   - Test with a complex prompt
   - If test fails, revert from backup
   - Verify bridge still binds to 127.0.0.1

4. If human rejects patch:
   - Document decision
   - Continue using direct bridge workaround

Stop Conditions:
- Bridge bind changes to 0.0.0.0
- Patch breaks bridge functionality
- Secrets exposed
- Agent Zero authority changed

Expected Output:
- Patch decision (approved/rejected)
- Backup path if patched
- Test result
- Revert status if needed

Note:
This is a future optional prompt. Do not execute unless explicitly requested.
