- [ ] Human has read docs/phase12/PHASE12B2_AGENT_ZERO_LOCAL_GEMMA_PROVIDER_REVIEW.md
- [ ] Human understands that 10.0.2.2 is the slirp4netns gateway for Podman
- [ ] Human understands that first model load may take >30 seconds
- [ ] Human explicitly approves this config patch

- Change ONLY the Ollama provider base URL to http://10.0.2.2:11434.

- Do NOT grant repo write authority.

2. Create one chat context.

Revert Plan:

1. Stop Agent Zero.
2. Restore backed-up config.
