- Read docs/integrations/ruvector/RUVECTOR_INTEGRATION_PLAN.md

5. Check compatibility:

   - Build-from-source requirements

6. Write assessment report to:
   - docs/integrations/ruvector/RUVECTOR_ASSESSMENT_REPORT.md

7. Update integration plan with findings:
   - docs/integrations/ruvector/RUVECTOR_INTEGRATION_PLAN.md

- Do not install Rust/npm dependencies
- Do not build or run RuVector
