# Phase 7A.2: A0 CLI Install and Endpoint Fix

**Goal:** Install A0 CLI user-locally and verify connectivity to Agent Zero.

**When to run this prompt:**
- Only if Phase 7A.1 report says `ready_for_a0_cli_install`
- After reviewing the installer safety analysis

---

## Hard Boundaries (Same as Phase 7A.1)

- Do not use sudo.
- Do not install packages system-wide.
- Do not use host networking.
- Do not inspect .env contents or secrets.
- Do not modify firewall, USBGuard, ClamAV, Lynis, rpm-ostree, systemd timers, Ollama config, or OpenCode permissions.
- Do not create system services or autostart entries.
- Do not leave Agent Zero running at the end unless explicitly needed and documented.
- Keep generated reports in ~/projects/gem/docs or canonical paths.

---

## Allowed Actions

- Run the downloaded installer (user-local, no sudo):
  ```bash
  cd ~/tmp/a0-cli-inspect
  bash install.sh
  ```
- OR perform equivalent manual install if documented
- Use `AGENT_ZERO_HOST` env var if endpoint doesn't work
- Start Agent Zero with existing helper script
- Verify endpoint with simple curl or A0 CLI
- Run only harmless connector check (e.g., `a0 --version`, `a0 --help`)
- Stop Agent Zero at end unless intentionally left running for manual UI review

---

## Tasks

### 1. Install A0 CLI

```bash
cd ~/tmp/a0-cli-inspect
bash install.sh
```

Or verify equivalent manual method if installer has issues.

### 2. Verify Installation

```bash
which a0
a0 --version
a0 --help
ls -la ~/.local/bin/a0
```

### 3. Start Agent Zero

```bash
agent-zero-up
sleep 8
agent-zero-status
```

### 4. Test Connectivity

**Option A - Direct curl (may still fail due to networking):**
```bash
curl -sS --max-time 10 http://127.0.0.1:5080 | head -20
```

**Option B - Use AGENT_ZERO_HOST (try first):**
```bash
export AGENT_ZERO_HOST=http://127.0.0.1:5080
a0 --help  # or simple test command
```

**Option C - Internal exec (always works):**
```bash
podman exec agent-zero curl -sS http://localhost:80/ | head -20
```

### 5. Simple Connector Check

```bash
# Test if A0 CLI can reach Agent Zero (if endpoint works)
a0 version 2>&1 || echo "CLI cannot reach Agent Zero"
```

**Note:** Do NOT run actual agent spawns. This is just connectivity verification.

### 6. Stop Agent Zero

```bash
agent-zero-down
agent-zero-status
```

Unless explicitly needed for manual UI review, stop the container.
