## Example: Bazzite Query

```markdown
# Memory Query Output

## Query
- **Query string:** "What firewall tool does Bazzite use?"
- **Date:** 2026-04-30
- **Scope:** approved sources only

## RuVector Results

### Semantic Matches
| Rank | Source | Relevance Score |
|------|--------|------------------|
| 1 | ~/.local/share/bazzite-security/gemma-knowledge/docs/FINAL_POLICY.md | 0.92 |
| 2 | ~/.local/share/bazzite-security/gemma-knowledge/index/chunks.jsonl | 0.87 |
| 3 | ~/.local/share/bazzite-security/gemma-knowledge/docs/OPERATIONS.md | 0.81 |

### Key Excerpts
- FINAL_POLICY.md: "Bazzite uses firewalld, not ufw"
- OPERATIONS.md: "Configure firewall with firewall-cmd"

## Stage 3A Fallback Results

### Deterministic Matches
| Rank | Source | Keywords Matched |
|------|--------|------------------|
| 1 | FINAL_POLICY.md | firewall, Bazzite |
| 2 | OPERATIONS.md | firewall, configure |

### Key Excerpts
- FINAL_POLICY.md: "Bazzite uses firewalld, not ufw"
- OPERATIONS.md: "Use firewall-cmd for configuration"

## Comparison

### Agreement
- **High** - semantic and deterministic results agree fully

### Points of Agreement
- Both sources cite firewalld
- Both reference firewall-cmd
- Consistent policy information

### Points of Difference
- None - perfect agreement on this query

### Quality Assessment
- RuVector provided similar results to Stage 3A
- Both returned the same authoritative source

## Sources Used

| Source | Type | Used By |
|--------|------|---------|
| FINAL_POLICY.md | Approved | Both |
| OPERATIONS.md | Approved | Both |

## Gemma Synthesis

[Not requested for this query]

## Uncertainty

### Ambiguous Areas
- None

### Recommendations
- None needed - clear answer available

## Stop Conditions Triggered

- [ ] Query touched denied paths
- [ ] RuVector unavailable - used Stage 3A only
- [ ] Results fundamentally disagree - escalated to human
- [x] None triggered

## Recommended Next Step

- **Use either RuVector or Stage 3A** - Both return consistent, authoritative answer about firewalld

## Metadata

- **Query ID:** mem-20260430-001
- **Workflow:** Semantic Context Lookup
- **Generated by:** Documentation
- **Reviewed by:** pending
```
